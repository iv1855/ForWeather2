
Активность 1:
Список городов.
Поля интерфейса:
- наименование города (ввод букв или выбор из списка)
- ИД (из СУБД)
- Число записей в СУБД
Функции:
- добавить город
- удалить город (и связанные данные в БД)
- при клике на город вызов активности с данными о погоде

Активность 2:
Данные погоды по городу.
Поля интерфейса:
- вверху наименование и ИД города
Список из данных:
- дата
- температура
- описание
Функции:
 - если в СУБД нет текущей даты, то автоматиески подгружается из интернета


 Активность 3:
 Панель управления.
 Поля интерфейса:
 - Город
 - ИД
 - кнопка Добавить город
 Функции: (по указанному городу)
  - загрузить в БД даные за сегодня
  - загрузить в БД даные за пред. 3 дня
  - очистить все данные
