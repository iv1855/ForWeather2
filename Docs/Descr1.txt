
Активность 1:
Список городов.(Основная)
Поля интерфейса:
- наименование города (ввод букв или выбор из списка)
- ИД (из СУБД)
- Число записей в СУБД
Функции:
- добавить город
- удалить город (и связанные данные в БД)
- при клике на город вызов активности с данными о погоде
- перейти в панель управления

Активность 2:
Данные погоды по городу.
Поля интерфейса:
- вверху наименование и ИД города
Список из данных:
- дата
- температура
- описание
Функции:
 - если в СУБД нет текущей даты, то автоматически подгружается из интернета


 Активность 3:
 Панель управления.
 Поля интерфейса:
 - Город
 - ИД
 - кнопка Добавить город
 Функции: (по указанному городу)
  - загрузить в БД данные за сегодня
  - загрузить в БД данные за пред. 3 дня
  - очистить все данные
